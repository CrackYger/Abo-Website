# Gianni Abo‑Portal (Split Version)

Empfohlenes Setup mit klarer Struktur (`components/`, `lib/`, `tests/`) und Vitest‑Unit‑Tests.

## Quickstart
```bash
npm i
npm run dev
```
Öffne dann die angezeigte `localhost`‑URL.

## Tests
```bash
npm run test
```

## Konfiguration
Passe `src/constants.js` an (E‑Mail und Admin‑PIN).
